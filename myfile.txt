4. Patching Activity : 

For every three months will do pacthing for preprod and production servers.
Will open new cr for patching activity and will get it approved for the CR from DCM and Manager as well application team.
Once we got approved all the stakeholders as per schedule.
Vmware team take the snapshot of the scheduled servers. 
we collect the prechecks like kernal version,crontabs and application jar,war versions will disbale the alerts
and we inform the application team stopping the application like jars.
we run the yum update command for patching.
yum update --exclude=openjdk
After pacthing we reboot the servers and will verify post checks like kernal versions,os versions, fstab entries 
Everything looks good will start the application and will inform application team to check the functionality with help testing team.
